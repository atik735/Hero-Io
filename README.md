
# 📱 Ultra Apps

Ultra Apps is a modern web-based app store where users can explore, search, and discover useful applications in one place. The platform provides a smooth and responsive user experience with real-time search functionality and a clean UI.

Users can browse all available apps, view detailed information, and quickly find what they need through an intuitive interface. The project focuses on performance, simplicity, and usability.

# 🚀 Features

🔍 Live search (case-insensitive filtering)

📊 Display total number of apps

📱 Responsive design for all devices

⚡ Fast loading with optimized performance

🎨 Clean and modern UI

❌ “No App Found” state when search has no results

# 🛠️ Technologies Used

- ⚛️ React
- 🔀 React Router
- ⚡ Vite
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 🎯 React Icons
- 🔔 React Toastify
